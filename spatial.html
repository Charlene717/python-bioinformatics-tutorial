二、Squidpy & SpatialData

Squidpy = 在 AnnData 上加空間分析 (鄰居圖、Moran's I、空間共表現)。

SpatialData = 新一代資料模型，整合表達矩陣 + 影像 + 形狀 (cell segmentation polygons)。

2025 年的趨勢：scanpy 的舊空間 tutorial 被 deprecated，新分析統一用 squidpy + spatialdata。

Squidpy = AnnData-aware spatial analysis (neighbor graphs, Moran's I, spatial co-expression).

SpatialData = next-gen data model unifying expression + images + shapes (cell segmentation polygons).

2025 trend: scanpy's old spatial tutorial is deprecated; new analysis uses squidpy + spatialdata.

完整流程

三、Visium 完整範例

import scanpy as sc, squidpy as sq

# 從 SpaceRanger 輸出讀入 / Read from SpaceRanger output
adata = sc.read_visium("data/raw/visium_brain/")
adata.var_names_make_unique()
# adata.obsm["spatial"] = (n_spots, 2) 像素座標
# adata.uns["spatial"][library]["images"] = HE 影像
print(adata)

# 在組織切片上看 QC
adata.var["mt"] = adata.var_names.str.startswith("MT-")
sc.pp.calculate_qc_metrics(adata, qc_vars=["mt"], inplace=True)

sc.pl.spatial(adata, color=["total_counts","n_genes_by_counts","pct_counts_mt"],
              cmap="viridis")

# 過濾：spot 太空、太多 mt
sc.pp.filter_cells(adata, min_counts=500)
adata = adata[adata.obs.pct_counts_mt < 20].copy()
# 跟 scRNA 一樣跑 normalize → HVG → PCA → UMAP → Leiden
adata.layers["counts"] = adata.X.copy()
sc.pp.normalize_total(adata)
sc.pp.log1p(adata)
sc.pp.highly_variable_genes(adata, n_top_genes=2000, flavor="seurat_v3",
                            layer="counts")
sc.pp.scale(adata, max_value=10)
sc.tl.pca(adata)
sc.pp.neighbors(adata)
sc.tl.umap(adata)
sc.tl.leiden(adata, resolution=1.0, key_added="clusters")

# 把 cluster 投回組織切片！ / Project clusters back onto the tissue
sc.pl.spatial(adata, color="clusters")

# 看特定基因的空間分佈
sc.pl.spatial(adata, color=["HPCA","TTR"], cmap="magma")
# 1. 建空間鄰居圖 / Build spatial neighbor graph
sq.gr.spatial_neighbors(adata, coord_type="generic", n_neighs=6)

# 2. Moran's I — 找空間自相關 (spatially variable) 基因
sq.gr.spatial_autocorr(adata, mode="moran", genes=adata.var_names[:3000])
top_svg = adata.uns["moranI"].head(10)
print(top_svg)

# 視覺化 top SVG
sc.pl.spatial(adata, color=top_svg.index[:6], ncols=3)

# 3. 鄰里富集分析 / Neighborhood enrichment
# 哪些 cluster 在空間上「常常相鄰」？
sq.gr.nhood_enrichment(adata, cluster_key="clusters")
sq.pl.nhood_enrichment(adata, cluster_key="clusters")

# 4. 共出現分析 / Co-occurrence
sq.gr.co_occurrence(adata, cluster_key="clusters")
sq.pl.co_occurrence(adata, cluster_key="clusters", clusters="3")

# 5. Ligand-receptor 分析 (LIANA)
# pip install liana
import liana as li
li.method.cellphonedb(adata, groupby="clusters", resource_name="consensus",
                      use_raw=False)
li.pl.dotplot(adata, colour="lr_means", size="specificity_rank")
# Xenium / MERFISH 是真正單細胞解析度
# Xenium / MERFISH are true single-cell resolution
import spatialdata as sd
import spatialdata_io as sd_io

# 讀入 Xenium 完整資料 (含 cell segmentation, 多解析度影像)
sdata = sd_io.xenium("data/raw/xenium_breast/")
print(sdata)
# SpatialData object: tables (cell × gene), shapes (polygons),
#                    images (DAPI, IF), points (transcripts)

# 取出 AnnData 給 scanpy 用
adata = sdata.tables["table"]

# 同樣的 scanpy + squidpy 流程，但解析度是真正單細胞
sq.gr.spatial_neighbors(adata, coord_type="generic",
                        delaunay=True)              # Delaunay 三角法
sq.gr.spatial_autocorr(adata, mode="moran")

關鍵分析

四、空間獨特分析

📍 空間變異基因 (SVG)

不只是「在 cluster 間有差」，而是「在空間上有結構」的基因。
方法：Moran's I (squidpy 內建)、SpatialDE、SPARK-X。

Genes whose expression varies structurally over space, not just between clusters.
Methods: Moran's I (squidpy built-in), SpatialDE, SPARK-X.

🧫 細胞類型解構

Visium 一個 spot 是混合細胞，需用 scRNA 參考圖譜分解出每 spot 的細胞組成。
工具：cell2location、RCTD、Tangram、Stereoscope。

Each Visium spot is a cell mixture; deconvolve into cell-type composition using a scRNA reference.
Tools: cell2location, RCTD, Tangram, Stereoscope.

🔁 微環境分析

哪些細胞類型常常出現在彼此附近？
squidpy nhood_enrichment + co_occurrence。

Which cell types appear near each other in space?
squidpy nhood_enrichment + co_occurrence.

📐 影像對齊

把連續切片或多染色 IHC 的影像對齊到同一座標系。
工具：SpatialData + STalign / PASTE。

Align serial sections or multi-stain IHC into the same coordinate system.
Tools: SpatialData + STalign / PASTE.
