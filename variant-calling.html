變異呼叫 & VCF

從 BAM 到突變列表 — pipeline 概念、VCF 格式、用 cyvcf2 / pysam 解析。

From BAM to a list of mutations — pipeline overview, VCF format, parsing with cyvcf2 / pysam.

概念

一、變異的種類

類型意義呼叫工具
SNP / SNV單個 base 的取代（最常見）GATK HaplotypeCaller, bcftools, DeepVariant Single-base substitution (most common)
Indel小段（<50 bp）插入或刪除GATK, bcftools, FreeBayes Small (<50 bp) insertion or deletion
SV結構變異（>50 bp 重排、倒置、易位）Manta, DELLY, Lumpy Structural variant (>50 bp rearrangement)
CNV複本數變異（區段倍增 / 缺失）CNVkit, GATK CNV, Control-FREEC Copy number variant
Somatic mutation癌細胞特有的突變（vs 正常組織）Mutect2, Strelka, VarScan Tumor-specific mutation (vs normal tissue)
工作流

二、標準工作流

原始資料 QC

FastQC + fastp / trim_galore 修剪 adapter、低品質端。

FastQC + fastp / trim_galore for adapter and low-Q trimming.

比對到參考基因組

BWA-MEM2 (DNA) 對到 hg38。輸出 BAM。

BWA-MEM2 (DNA) → hg38 → BAM.

處理 BAM

排序 (samtools sort) → 標記重複 (Picard MarkDuplicates) → BQSR (GATK BaseRecalibrator)。

Sort (samtools sort) → MarkDuplicates (Picard) → BQSR (GATK BaseRecalibrator).

呼叫變異

GATK HaplotypeCaller 產生 g.VCF；多樣本聯合 → GenomicsDB → GenotypeGVCFs → cohort VCF。

GATK HaplotypeCaller → g.VCF; joint calling → GenomicsDB → GenotypeGVCFs → cohort VCF.

過濾與註解

VQSR / hard filters → VCF 過濾 → SnpEff / VEP / ANNOVAR 加上基因/影響註解。

VQSR / hard filters → SnpEff / VEP / ANNOVAR for gene/effect annotation.

篩選稀有變異、查 ClinVar / OMIM / gnomAD、做 GWAS / 連鎖分析。

Filter rare variants, cross-check ClinVar / OMIM / gnomAD, GWAS / linkage.

VCF 格式

三、VCF 檔案結構

##fileformat=VCFv4.2 ##INFO=<ID=DP,Number=1,Type=Integer,Description="Total depth"> ##FORMAT=<ID=GT,Number=1,Type=String,Description="Genotype"> ##FORMAT=<ID=AD,Number=R,Type=Integer,Description="Allelic depths"> #CHROM POS ID REF ALT QUAL FILTER INFO FORMAT Sample1 chr1 14653 rs76343445 C T 99 PASS DP=53;AF=0.5 GT:AD:DP 0/1:25,28:53 chr1 14671 . G A 63 PASS DP=42 GT:AD:DP 1/1:0,42:42
  • 標頭 (##)：定義 INFO/FORMAT/FILTER 欄位的意義。
  • 主表 8+ 欄：CHROM/POS/ID/REF/ALT/QUAL/FILTER/INFO 後面接 FORMAT + 每個樣本的值。
  • GT (基因型)： 0/0=純合野生、0/1=異合、1/1=純合突變、1/2=兩種不同 ALT。
  • AD (allelic depth)：每個 allele 的 read 數。
  • DP：該位點總深度。
  • Header (##): defines INFO/FORMAT/FILTER fields.
  • Body 8+ columns: CHROM/POS/ID/REF/ALT/QUAL/FILTER/INFO + FORMAT + per-sample values.
  • GT (genotype): 0/0 hom ref, 0/1 het, 1/1 hom alt, 1/2 two different ALTs.
  • AD (allelic depth): reads supporting each allele.
  • DP: total depth at the site.

# cyvcf2: 純 C 寫，速度比 PyVCF 快 50 倍以上
# cyvcf2: written in C, >50× faster than PyVCF

from cyvcf2 import VCF

vcf = VCF("data/raw/cohort.vcf.gz")
print(vcf.samples)               # 樣本列表

for v in vcf:
    print(v.CHROM, v.POS, v.REF, v.ALT,
          v.QUAL, v.INFO.get("DP"),
          v.gt_types)              # 0=hom ref, 1=het, 2=hom alt, 3=missing
    break

# 抓特定區段 (要有 .tbi index)
for v in vcf("chr17:7670000-7690000"):
    print(v.POS, v.gt_quals)

# 過濾：只保留高品質、PASS、QD 高的
filtered = (v for v in vcf
            if v.FILTER is None           # 'PASS'
            and v.QUAL > 30
            and v.INFO.get("QD", 0) > 2)
import pysam

vcf = pysam.VariantFile("data/raw/cohort.vcf.gz")
for rec in vcf:
    print(rec.contig, rec.pos, rec.ref, rec.alts,
          rec.qual, dict(rec.info))
    for sample, sv in rec.samples.items():
        print(sample, sv["GT"], sv.get("AD"), sv.get("DP"))
    break

# 區段查詢
for rec in vcf.fetch("chr17", 7670000, 7690000):
    print(rec.pos)
# 簡單分析時，把 VCF 用 pandas 讀 (略過 ## header)
import pandas as pd

# 找出 #CHROM 那行的位置
def read_vcf(fp):
    with open(fp) as f:
        for i, line in enumerate(f):
            if line.startswith("#CHROM"):
                header = line.lstrip("#").strip().split("\t")
                return pd.read_csv(fp, sep="\t", comment="#",
                                   names=header)
df = read_vcf("sample.vcf")
print(df[["CHROM","POS","REF","ALT","QUAL","FILTER"]].head())
實戰範例

五、找 TP53 稀有有害變異

from cyvcf2 import VCF
import pandas as pd

vcf = VCF("data/raw/cohort.snpeff.vcf.gz")

rows = []
for v in vcf("chr17:7670000-7690000"):              # TP53 region
    if v.FILTER is not None: continue            # 只留 PASS
    af = v.INFO.get("AF") or v.INFO.get("gnomAD_AF") or 0
    if af and af > 0.01: continue                    # 排除常見變異

    ann = v.INFO.get("ANN", "").split(",")
    if not ann: continue
    fields = ann[0].split("|")
    impact = fields[2] if len(fields) > 2 else ""
    gene   = fields[3] if len(fields) > 3 else ""
    effect = fields[1] if len(fields) > 1 else ""

    if impact in ("HIGH", "MODERATE") and gene == "TP53":
        rows.append({
            "chr": v.CHROM, "pos": v.POS,
            "ref": v.REF, "alt": ",".join(v.ALT),
            "effect": effect, "impact": impact, "AF": af,
            "qual": v.QUAL,
        })

df = pd.DataFrame(rows).sort_values("pos")
df.to_csv("results/tp53_rare.tsv", sep="\t", index=False)
print(df)
