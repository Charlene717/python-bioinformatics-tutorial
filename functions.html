函式、類別、模組

把分散的程式碼包成可重用的單位 — 寫得乾淨、改起來不哭。

Package scattered code into reusable units — write clean, refactor without tears.

函式

一、函式 (def)

函式 = 一個有名字的程式碼區塊，可以被重複呼叫。原則：「同一段邏輯出現第三次，就抽成函式」。

A function = a named block of code you can call repeatedly. Rule of thumb: "If the same logic appears a third time, extract it into a function."

def gc_content(seq):
    """一行說明 / One-line description"""
    seq = seq.upper()
    gc = seq.count("G") + seq.count("C")
    return gc / len(seq) * 100

# 呼叫 / Call
gc = gc_content("ATGCGT")         # 50.0

# 一個函式可以回傳多個值 (其實是 tuple)
# Multiple return values (it's actually a tuple)
def seq_stats(seq):
    return len(seq), gc_content(seq)

length, gc = seq_stats("ATGCGT")    # tuple unpacking
# 預設值 / Default values
def qc_filter(adata, min_genes=200, max_mt=10):
    return adata[(adata.obs.n_genes_by_counts > min_genes) &
                 (adata.obs.pct_counts_mt < max_mt)]

qc_filter(adata)                      # 用預設

# *args 任意多個位置參數 / Variable positional args
def merge_lists(*lists):
    out = []
    for l in lists: out.extend(l)
    return out
merge_lists(["A"], ["B","C"], ["D"])

# **kwargs 任意多個關鍵字參數 / Variable keyword args
def sample_meta(**kwargs):
    for k, v in kwargs.items():
        print(f"{k} = {v}")
sample_meta(donor="D1", age=35, condition="WT")

# 危險：可變預設值！ / Mutable default trap
def bad(x, hits=[]):     # ❌ list 在函式定義時建立一次
    hits.append(x); return hits
def good(x, hits=None):   # ✅ 用 None 當哨兵
    if hits is None: hits=[]
    hits.append(x); return hits
from typing import list, tuple, Optional

def gc_content(seq: str) -> float:
    """Calculate GC content of a DNA sequence.

    Parameters
    ----------
    seq : str
        DNA sequence (case-insensitive). May contain N.

    Returns
    -------
    float
        GC percentage in 0–100. Returns 0.0 if seq is empty.

    Examples
    --------
    >>> gc_content("ATGC")
    50.0
    """
    if not seq: return 0.0
    seq = seq.upper()
    return (seq.count("G") + seq.count("C")) / len(seq) * 100

# type hint (3.10+) 可以用內建型別 list[str] 而不用 List[str]
def top_genes(scores: dict[str, float], k: int = 10) -> list[str]:
    return sorted(scores, key=scores.get, reverse=True)[:k]

# Type hints are documentation; they don't affect runtime

變數作用域

二、變數作用域 (LEGB)

Python 找變數時依序看：Local → Enclosing → Global → Built-in。理解這個能避免「為什麼我的變數突然不見了」這類詭異 bug。

Python looks up names in order: Local → Enclosing → Global → Built-in. Knowing this avoids "where did my variable go?" bugs.

x = 10                    # global

def outer():
    x = 20                # enclosing (對 inner 而言)
    def inner():
        x = 30            # local
        print(x)         # 30 (Local first)
    inner()
    print(x)             # 20

outer()
print(x)                 # 10 (global 沒被改)

# 想在函式裡修改 global，要用 global 關鍵字
# To modify a global from inside, use 'global'
counter = 0
def increment():
    global counter
    counter += 1           # 沒寫 global 會出 UnboundLocalError

# 但通常更好的做法是：用回傳值，不要改 global
# Better: return values, avoid modifying globals
模組與匯入

三、模組與 import

每個 .py 檔就是一個模組 (module)；一個資料夾裡放多個 .py + __init__.py 就是一個套件 (package)。

Each .py file is a module; a folder of .py files plus __init__.py is a package.

類別

四、類別 (class)

類別是「自訂型別」。在生資領域，pandas DataFrame、AnnData、Seq 都是類別。你大部分時間是使用別人寫好的 class，但偶爾自己寫個簡單的也很有用。

A class defines a custom type. In bioinformatics, DataFrame, AnnData, and Seq are all classes. You'll spend most time using classes others wrote, but writing simple ones yourself can be very useful.

class Sample:
    def __init__(self, sample_id, condition, n_cells=0):
        # self = 這個物件本身 / self = the instance
        self.sample_id = sample_id
        self.condition = condition
        self.n_cells = n_cells

    def summary(self):
        return f"{self.sample_id} ({self.condition}): {self.n_cells} cells"

    def __repr__(self):
        # 控制 print() 的輸出 / Controls how print() displays the obj
        return f"<Sample {self.sample_id}>"

s = Sample("PBMC1", "WT", n_cells=5000)
print(s.summary())     # PBMC1 (WT): 5000 cells
print(s)               # <Sample PBMC1>
