Pandas 資料框

處理 metadata、樣本表、DE 結果、count matrix —— 生資資料分析的瑞士刀。

Metadata, sample sheets, DE results, count matrices — the Swiss Army knife of bioinformatics data wrangling.

概念

一、Series 與 DataFrame

Series = 一維帶標籤陣列（像 dict + ndarray 的混合體）。
DataFrame = 二維表格，每欄是一個 Series，欄有名字、列有 index。

生資場景：sample sheet（樣本 metadata）、DE 結果表、基因註解都是 DataFrame；某 cell 跨所有基因的表達量、某基因的 p-value 跨所有比較組都是 Series。

Series = labeled 1-D array (a dict + ndarray hybrid).
DataFrame = 2-D table, each column a Series with a name and a row index.

Bioinformatics: sample sheets, DE results, gene annotations are all DataFrames; a single cell's expression across all genes is a Series.

四、選資料

📌
核心區別： .loc[] 用標籤名選；.iloc[] 用整數位置選。混用是新手 bug 大宗 — 永遠寫明哪個。 Core distinction: .loc[] selects by label; .iloc[] selects by integer position. Mixing them is a top source of bugs — always be explicit. 
df["condition"]              # 一欄 → Series
df[["condition", "n_cells"]]  # 多欄 → DataFrame (注意雙層方括號)

# 新增/修改一欄
df["qc_status"] = np.where(df["mt_pct"] < 10, "PASS", "FAIL")
df["log_cells"] = np.log1p(df["n_cells"])

# 用 assign 做鏈式風格 / Chainable assign
df = (df
      .assign(log_cells=lambda d: np.log1p(d.n_cells))
      .assign(passed=lambda d: d.mt_pct < 10))

# 刪欄 / Drop columns
df = df.drop(columns=["qc_status", "log_cells"])
# .loc[列標籤, 欄標籤]  → 用名字
df.loc["S1"]                       # sample_id == "S1" 那列
df.loc["S1", "condition"]          # 一格
df.loc[["S1","S3"], ["condition","mt_pct"]]
df.loc["S1":"S3"]                  # 連續範圍 (含右端！)

# .iloc[列位置, 欄位置] → 用整數
df.iloc[0]                # 第 0 列
df.iloc[0, 1]             # 第 0 列、第 1 欄
df.iloc[:3, :2]           # 前 3 列、前 2 欄 (不含右端)
df.iloc[-1]               # 最後一列
# 布林過濾 — 跟 NumPy 一樣，用 & | ~
df[df["mt_pct"] < 10]
df[(df.condition == "WT") & (df.n_cells > 5000)]
df[df.condition.isin(["WT", "KO"])]
df[~df.condition.isin(["control"])]   # ~ 表示 NOT

# 字串條件 / String filtering
df[df["sample_id"].str.startswith("PBMC")]
df[df["sample_id"].str.contains("_2024")]
df[df["sample_id"].str.match(r"^S\d+$")]    # regex
# query — 寫起來更乾淨 / Cleaner syntax
df.query("mt_pct < 10 and condition == 'WT'")
df.query("n_cells > @threshold")         # @ 引用 Python 變數

# 鏈式 (method chaining) — pandas 風格 / Pandas style
result = (df
    .query("mt_pct < 10")
    .assign(rank=lambda d: d.n_cells.rank())
    .sort_values("rank")
    .head(10)

生資範例

六、整合範例

import pandas as pd

# 讀 DESeq2 / edgeR 輸出的 TSV
de = pd.read_csv("results/de_results.tsv", sep="\t")

# 1. 看一眼結構
print(de.shape, de.dtypes)
print(de.head())

# 2. 計算 -log10(padj) 用於 volcano plot
de["neg_log_padj"] = -np.log10(de["padj"].replace(0, 1e-300))

# 3. 標記顯著上下調 (用 numpy.select 寫 if/elif)
conditions = [
    (de.padj < 0.05) & (de.log2FoldChange > 1),
    (de.padj < 0.05) & (de.log2FoldChange < -1),
]
choices = ["UP", "DOWN"]
de["sig"] = np.select(conditions, choices, default="NS")

# 4. 各組的數量
print(de.sig.value_counts())

# 5. Top 20 上調基因，依 fold change 排序
top_up = (de
    .query("sig == 'UP'")
    .nlargest(20, "log2FoldChange")
    .loc[:, ["gene","log2FoldChange","padj"]])

# 6. 跟 GTF 註解合併 — 加入 chromosome / biotype
annot = pd.read_csv("data/raw/annotation.tsv", sep="\t")
top_annot = pd.merge(top_up, annot, on="gene", how="left")

# 7. 輸出
top_annot.to_csv("results/top_up_annotated.tsv", sep="\t", index=False)
