開始

一、第一次執行 Python

Python 有三種執行方式，先弄懂差別：

  1. 互動式 (REPL)：終端機輸入 python 直接打程式碼，按 Enter 立刻看到結果。適合測試小片段。離開：exit() 或 Ctrl+D。
  2. 腳本 (.py 檔)：把程式存成 hello.py，終端機執行 python hello.py。適合正式分析、重複執行。
  3. Jupyter Notebook (.ipynb)：把程式分成 cell，可以邊寫邊看結果、夾雜文字說明。適合探索、論文圖。

Python runs in three different ways — get the differences straight:

  1. Interactive (REPL): type python in your terminal and enter code line by line; results appear instantly. Great for testing snippets. Exit with exit() or Ctrl+D.
  2. Scripts (.py file): save code as hello.py and run python hello.py. For formal, repeatable analyses.
  3. Jupyter Notebook (.ipynb): code split into cells, write-and-see-as-you-go with markdown notes. Great for exploration and paper figures.

變數與型別

二、變數、型別與基本運算

Python 不需要宣告型別 — 用 = 賦值即可。型別是動態的（同一個變數名可以再被指向不同型別）。

Python doesn't require type declarations — assign with = and you're done. Types are dynamic: the same name can later refer to a value of any type.

容器

三、容器：list / dict / tuple / set

📋 list — 有序、可變

用方括號 [ ]，最常用的容器。可以新增、修改、排序。

Created with square brackets [ ] — the workhorse container. You can append, modify, sort.

🗂️ dict — 鍵值對

用大括號 { } 加 :。透過鍵 (key) 取值 (value)，O(1) 超快查找。生資最常用！

Curly braces { } with :. Look up values by key in O(1). The most-used container in bioinformatics!

🔒 tuple — 不可變

用圓括號 ( )。無法修改，但可當 dict 的 key（list 不行）。常用於座標或回傳多值。

Round brackets ( ). Cannot be modified but can serve as a dict key (lists cannot). Often for coordinates or multi-value returns.

🎯 set — 無序、唯一

用大括號 { }（無 key）。自動去重，超快做交集/聯集 — 比較兩組基因清單必備。

Curly braces { } (no key). Auto-deduplicates, fast set ops — essential for comparing two gene lists.

五、整合範例：手寫 GC content + 反向互補

把上面學到的東西組合起來 — 不依賴 Biopython，純 Python 完成兩個經典操作。

Combine everything above — no Biopython, pure Python doing two classic operations.

def gc_content(seq):
    """回傳 GC% (0–100) / Return GC% (0–100)"""
    seq = seq.upper()
    gc = seq.count("G") + seq.count("C")
    return gc / len(seq) * 100

def reverse_complement(seq):
    """DNA 反向互補 / Reverse complement of DNA"""
    pairs = {"A": "T", "T": "A", "G": "C", "C": "G", "N": "N"}
    # 用 list comprehension + reverse + join，一行搞定
    return "".join(pairs[b] for b in seq.upper()[::-1])

# 測試 / Test
seq = "ATGCGTAATCG"
print(f"GC: {gc_content(seq):.1f}%")        # GC: 54.5%
print(f"RevComp: {reverse_complement(seq)}")  # CGATTACGCAT

# [::-1] 是 Python 慣用語：把字串/list 反轉
# [::-1] is the Pythonic idiom for reversing a sequence
