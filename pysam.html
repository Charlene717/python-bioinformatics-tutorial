CIGAR

三、CIGAR 字串

76M = 連續 76 個 base 都是 match/mismatch；50M2I20M = 50 match + 2 insertion + 20 match。常見符號：

  • M = alignment match (可能有 mismatch！)
  • I = insertion 在 read 上（read 有但 ref 沒有）
  • D = deletion 從 read 上（ref 有但 read 沒有）
  • S = soft-clipped（read 兩端沒對到，但 SEQ 還在）
  • H = hard-clipped（被剪掉、SEQ 也沒了）
  • N = skipped region（RNA-seq 跨 intron 必有）

76M = 76 contiguous matches/mismatches; 50M2I20M = 50M + 2-base insertion + 20M. Common symbols:

  • M = alignment match (could include mismatches!)
  • I = insertion in the read (read has it, ref doesn't)
  • D = deletion from the read (ref has it, read doesn't)
  • S = soft-clipped (read end didn't align, SEQ kept)
  • H = hard-clipped (cut off, SEQ removed)
  • N = skipped region (always present in RNA-seq across introns)

import pysam, numpy as np
import matplotlib.pyplot as plt

bam = pysam.AlignmentFile("sample.bam", "rb")

# 一個基因 region 的 coverage profile
# Coverage profile across a region of interest
chrom, start, end = "chr17", 7670000, 7690000     # TP53 大概位置
length = end - start
counts = np.zeros(length, dtype=int)

# 用 pileup() 一個 base 一個 base 統計
for col in bam.pileup(chrom, start, end, truncate=True):

# 畫圖

ax.set_xlabel("Position (chr17)"); ax.set_ylabel("Coverage")
fig.savefig("results/figures/coverage.pdf")
