scRNA-seq 完整流程

從 10X 矩陣到細胞類型註解 — 一份可以原封不動跑的 Python pipeline。

From a 10X matrix to annotated cell types — a Python pipeline you can run as-is.

總覽

一、9 個步驟

QC & filter

計算 nFeature / nCount / pct_mt → 過濾低品質細胞與基因。

Compute nFeature / nCount / pct_mt → filter low-quality cells and genes.

Normalize + log1p

標準化每個 cell 到固定 size factor (10000) → log1p。

Normalize each cell to fixed size factor (10000) → log1p.

HVG selection

選擇高變異基因（2000–3000 個）做後續降維。

Select highly variable genes (2000–3000) for downstream.

Scale & PCA

z-score 後做 PCA，保留 ~50 個 PC。

Z-score then PCA, keep ~50 PCs.

Neighbors & UMAP

在 PCA 空間建近鄰圖 → UMAP 降到 2D。

kNN graph in PCA space → UMAP to 2D.

Clustering (Leiden)

用 Leiden 演算法在近鄰圖上分群。

Leiden community detection on the kNN graph.

Marker genes

每個 cluster 找代表基因（rank_genes_groups）。

Find marker genes per cluster (rank_genes_groups).

Cell type annotation

用 marker 對照已知細胞類型，或用自動工具 (CellTypist, SingleR)。

Match markers to known cell types, or use auto tools (CellTypist, SingleR).

DE / 進階

跨 condition 比較表達差異、軌跡分析、整合多樣本。

DE across conditions, trajectory, multi-sample integration.

完整程式

二、可執行 pipeline

import scanpy as sc
import numpy as np
sc.settings.verbosity = 3; sc.settings.set_figure_params(dpi=100, facecolor="white")

# --- Step 1: 載入 + QC ---
adata = sc.read_10x_mtx("data/raw/pbmc/filtered_feature_bc_matrix/",
                        var_names="gene_symbols", cache=True)
adata.var_names_make_unique()

# 標記 mt / ribo / hb 基因
adata.var["mt"]   = adata.var_names.str.startswith("MT-")
adata.var["ribo"] = adata.var_names.str.startswith(("RPS","RPL"))
adata.var["hb"]   = adata.var_names.str.contains("^HB[^P]")
sc.pp.calculate_qc_metrics(adata, qc_vars=["mt","ribo","hb"],
                           inplace=True, log1p=False)

# 視覺化 QC，先看再決定閾值！
sc.pl.violin(adata, ["n_genes_by_counts","total_counts","pct_counts_mt"],
             jitter=0.4, multi_panel=True)
sc.pl.scatter(adata, x="total_counts", y="n_genes_by_counts", color="pct_counts_mt")

# 過濾
sc.pp.filter_cells(adata, min_genes=200)
sc.pp.filter_genes(adata, min_cells=3)
adata = adata[(adata.obs.n_genes_by_counts < 6000) &
              (adata.obs.pct_counts_mt < 15)].copy()

# --- Step 2: Normalize + log1p ---
adata.layers["counts"] = adata.X.copy()      # 保留 raw counts
sc.pp.normalize_total(adata, target_sum=1e4)
sc.pp.log1p(adata)

# --- Step 3: HVG ---
sc.pp.highly_variable_genes(adata, n_top_genes=2000, flavor="seurat_v3",
                            layer="counts")
sc.pl.highly_variable_genes(adata)

# --- Step 4: Scale + PCA ---
adata.raw = adata           # 保留 lognorm 給後續視覺化用
adata = adata[:, adata.var.highly_variable]
sc.pp.scale(adata, max_value=10)
sc.tl.pca(adata, n_comps=50, svd_solver="arpack")
sc.pl.pca_variance_ratio(adata, n_pcs=50, log=True)
# --- Step 5: Neighbors + UMAP ---
sc.pp.neighbors(adata, n_neighbors=15, n_pcs=30)
sc.tl.umap(adata)

# --- Step 6: Clustering ---
sc.tl.leiden(adata, resolution=0.6, key_added="leiden")

# 視覺化
sc.pl.umap(adata, color=["leiden","total_counts","pct_counts_mt"],
           wspace=0.3, ncols=3)

# --- Step 7: Marker genes ---
sc.tl.rank_genes_groups(adata, "leiden", method="wilcoxon")

# Top 25 markers per cluster (DataFrame)
markers = sc.get.rank_genes_groups_df(adata, group=None)
top = (markers.query("pvals_adj < 0.01 and logfoldchanges > 1")
              .groupby("group").head(25))

# 各種 marker 視覺化 / Marker visualizations
sc.pl.rank_genes_groups_dotplot(adata, n_genes=5)
sc.pl.rank_genes_groups_heatmap(adata, n_genes=5, show_gene_labels=True)
sc.pl.umap(adata, color=["CD3D","CD8A","MS4A1","CD14","NKG7"], use_raw=True)
# --- Step 8: Manual 註解 (依 marker 知識) ---
cluster_to_celltype = {
    "0": "CD4 T",    "1": "CD14 Mono", "2": "B",
    "3": "CD8 T",    "4": "NK",         "5": "FCGR3A Mono",
    "6": "DC",       "7": "Platelet",
}
adata.obs["cell_type"] = adata.obs["leiden"].map(cluster_to_celltype)
sc.pl.umap(adata, color="cell_type", legend_loc="on data")

# 自動註解 / Automatic annotation (CellTypist)
# pip install celltypist
import celltypist
predictions = celltypist.annotate(adata,
    model="Immune_All_Low.pkl", majority_voting=True)
adata = predictions.to_adata()
sc.pl.umap(adata, color="majority_voting")

# --- DE 跨 condition ---
sc.tl.rank_genes_groups(adata, groupby="condition",
                        groups=["KO"], reference="WT")
de_df = sc.get.rank_genes_groups_df(adata, group="KO")
de_df.to_csv("results/de_KO_vs_WT.csv", index=False)

# 別忘了存檔！ / Don't forget to save
adata.write_h5ad("data/processed/pbmc_annotated.h5ad", compression="gzip")
# 多樣本整合 (移除 batch effect)
# Multi-sample integration (remove batch effect)
import scanpy as sc
import anndata as ad

# 讀入多個樣本，合併
adatas = [sc.read_h5ad(f"data/processed/{s}.h5ad") for s in ["S1","S2","S3"]]
for i, a in enumerate(adatas): a.obs["batch"] = f"batch{i}"
adata = ad.concat(adatas, join="outer", label="sample", keys=["S1","S2","S3"])

# Harmony — 最常用的整合方法 / Most popular integration
# pip install harmonypy
sc.external.pp.harmony_integrate(adata, key="batch", basis="X_pca",
                                  adjusted_basis="X_pca_harmony")
sc.pp.neighbors(adata, use_rep="X_pca_harmony")
sc.tl.umap(adata)
sc.tl.leiden(adata)

# scVI — 深度學習整合（推薦給大型資料集） / scVI deep-learning integration
# pip install scvi-tools
import scvi
scvi.model.SCVI.setup_anndata(adata, batch_key="batch", layer="counts")
model = scvi.model.SCVI(adata, n_layers=2, n_latent=30)
model.train(max_epochs=200)
adata.obsm["X_scVI"] = model.get_latent_representation()
sc.pp.neighbors(adata, use_rep="X_scVI")
sc.tl.umap(adata)
參數選擇

三、關鍵超參數

參數預設該怎麼選
n_top_genes (HVG)2000大資料 (>100k cells) → 3000–4000；小資料 → 2000；多樣本整合可降到 1500Large data (>100k cells) → 3000-4000; small → 2000; integration → 1500
n_pcs50看 elbow plot — 通常 20–50；簡單組織 15 就夠Check elbow plot — usually 20-50; simple tissues OK with 15
n_neighbors15小資料 10；大資料 (>100k) 30；要保留稀有族群 → 較小Small data 10; large (>100k) 30; preserve rare populations → smaller
resolution (Leiden)1.0沒有正確答案。試 0.3, 0.5, 0.8, 1.2，挑能對應到生物學意義的No right answer. Try 0.3, 0.5, 0.8, 1.2; pick one that matches biology
⚠️
勿盲信預設值！同一份資料用 resolution=0.3 可能 5 個 cluster、1.2 可能 20 個 cluster。先用低 resolution 看大族，再針對某個族高 resolution 細分（subset 後重跑 neighbors+leiden）。 Don't trust defaults blindly. Same data may yield 5 clusters at resolution=0.3 and 20 at 1.2. Use low resolution for the big picture, then sub-cluster a specific population at higher resolution (subset and re-run neighbors+leiden).
