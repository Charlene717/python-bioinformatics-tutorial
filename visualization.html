Matplotlib 基礎

二、Figure vs Axes

📌
核心觀念：Matplotlib 中 Figure = 整張畫布，Axes = 一個子圖。一個 Figure 可以有 1 個或多個 Axes。
新手常見混淆：plt.plot(...)（隱式狀態，不推薦） vs ax.plot(...)（推薦，可控、可組合）。 Core concept: in Matplotlib, Figure = the whole canvas, Axes = one subplot. A Figure can hold one or many Axes.
Common confusion: plt.plot(...) (implicit state, not recommended) vs ax.plot(...) (recommended, controllable, composable). 
import matplotlib.pyplot as plt
import numpy as np

# 建立 figure 和 axes
fig, ax = plt.subplots(figsize=(6, 4))   # 寬 6 高 4 inch

# 在 ax 上繪圖 (推薦的 OO 風格)
x = np.linspace(0, 10, 100)
ax.plot(x, np.sin(x), label="sin", color="#2563a6", lw=2)
ax.plot(x, np.cos(x), label="cos", color="#f59e0b", lw=2, ls="--")

# 元素
ax.set_xlabel("Time", fontsize=12)
ax.set_ylabel("Amplitude", fontsize=12)
ax.set_title("Trig functions", fontsize=14, fontweight="bold")
ax.legend(loc="upper right", frameon=False)
ax.spines["top"].set_visible(False)       # 移除上邊框
ax.spines["right"].set_visible(False)
ax.grid(alpha=0.3)

plt.tight_layout()
plt.show()
# 2 列 3 欄 = 6 個子圖
fig, axes = plt.subplots(2, 3, figsize=(12, 7))

# axes 是一個 2x3 的 ndarray，可以用 axes[0,1] 取
axes[0, 0].hist(np.random.normal(size=1000), bins=30)
axes[0, 0].set_title("Histogram")

axes[0, 1].scatter(np.random.rand(100), np.random.rand(100))
axes[0, 1].set_title("Scatter")

de["sig"] = np.where((de.padj < 0.05) & (de.log2FoldChange.abs() > 1),
                     np.where(de.log2FoldChange > 0, "UP", "DOWN"), "NS")

fig, ax = plt.subplots(figsize=(6,6))
colors = {"UP":"#c0392b", "DOWN":"#2563a6", "NS":"#bbb"}
for g, sub in de.groupby("sig"):

               c=colors[g], s=8, alpha=0.6, label=g)

# 標 top hits
top = de.nsmallest(10, "padj")
for _, r in top.iterrows():
    ax.annotate(r.gene, (r.log2FoldChange, -np.log10(r.padj)),
                fontsize=8, ha="left")

ax.axhline(-np.log10(0.05), color="gray", ls="--", alpha=.5)
ax.axvline(-1, color="gray", ls="--", alpha=.5)
ax.axvline(1, color="gray", ls="--", alpha=.5)
ax.set_xlabel("log$_2$ fold change"); ax.set_ylabel("-log$_{10}$(padj)")

顏色學

四、顏色選擇

📊 連續

單向漸層：表達量、深度、p-value。
推薦：viridis、magma、plasma — 對色盲友善、印灰階仍可辨。

Single-direction gradient: expression, depth, p-value.
Recommended: viridis, magma, plasma — colorblind-friendly, readable in greyscale.

↔️ 發散

有「中心」的資料：log2FC、相關係數、z-score。
推薦：RdBu_r、coolwarm、vlag。
務必設 center=0。

Data with a meaningful center: log2FC, correlations, z-scores.
Recommended: RdBu_r, coolwarm, vlag. Always set center=0.

🎨 類別

cluster ID、cell type。
推薦：<10 類用 tab10；10–20 類用 tab20；>20 類請考慮合併或用 scanpy.pl.palettes.default_102。

Cluster IDs, cell types.
Recommended: tab10 for <10 classes; tab20 for 10–20; for >20 consider merging or use scanpy.pl.palettes.default_102.

🚫
不要再用 jet/rainbow！它對色盲不友善、視覺不均勻、印灰階就毀了。Nature Methods 已正式建議改用 viridis 系列。 Stop using jet/rainbow! It's colorblind-hostile, perceptually non-uniform, and unreadable in greyscale. Nature Methods explicitly recommends viridis instead.
