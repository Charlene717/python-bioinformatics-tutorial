AnnData 與 scanpy

單細胞分析的核心資料結構與工具庫 — 看懂它，未來所有 scRNA / 空間 / 多體學工具都通了。

The core data structure and toolbox for single-cell analysis — master AnnData and the rest of scRNA / spatial / multi-omics opens up.

AnnData 結構

一、AnnData 是什麼？

AnnData = Annotated Data。一個 AnnData 物件代表一個 scRNA-seq（或任何高通量）實驗，包含：表達矩陣本身 + 每個 cell 的 metadata + 每個 gene 的 metadata + 各種降維結果。

AnnData = Annotated Data. One AnnData object represents one scRNA-seq (or any high-throughput) experiment: the expression matrix + per-cell metadata + per-gene metadata + various dimensionality reduction results.

# 視覺化結構 / Schema adata = AnnData(...) ├── .X # 表達矩陣 (n_cells × n_genes)，通常為 sparse ├── .obs # pandas DataFrame，每列一 cell，欄是 cell metadata (cluster, condition, donor) ├── .var # pandas DataFrame，每列一 gene，欄是 gene metadata (mt, hvg) ├── .obsm # dict of (n_cells × d) — 降維結果 X_pca, X_umap, X_tsne ├── .varm # dict of (n_genes × d) — 基因端的多維資料 ├── .obsp # dict of (n_cells × n_cells) — cell pair 距離 / connectivity ├── .uns # 任意 dict — 例如 'rank_genes_groups' 結果、palette └── .layers # dict of 同形狀矩陣 — counts, lognorm, scaled 共存
💡
記住這 7 個欄位，scanpy / squidpy / scvelo / cellrank 全部都圍著 AnnData 轉。看到 adata.obs[...] 就是在看 cell metadata；adata.obsm["X_umap"] 就是 UMAP 座標。 Memorize these 7 slots — scanpy / squidpy / scvelo / cellrank all revolve around AnnData. adata.obs[...] = cell metadata; adata.obsm["X_umap"] = UMAP coordinates.

四、sc.pp / sc.tl / sc.pl

🧹

sc.pp — preprocessing

修改 adata.X 或 adata.obs/var 的「資料前處理」函式：QC、normalize、HVG、scale、neighbors。

例：sc.pp.filter_cells、sc.pp.normalize_total、sc.pp.highly_variable_genes。

Functions that modify adata.X or metadata: QC, normalization, HVG, scaling, neighbors.

e.g. sc.pp.filter_cells, sc.pp.normalize_total, sc.pp.highly_variable_genes.

🔬

sc.tl — tools

分析工具：PCA、UMAP、Leiden、rank_genes、velocity、pseudotime。結果存進 adata.obsm/uns。

例：sc.tl.pca、sc.tl.umap、sc.tl.leiden、sc.tl.rank_genes_groups。

Analysis tools: PCA, UMAP, Leiden, marker genes, velocity, pseudotime. Results stored in adata.obsm/uns.

e.g. sc.tl.pca, sc.tl.umap, sc.tl.leiden, sc.tl.rank_genes_groups.

🎨

sc.pl — plotting

繪圖。每個 tool 大多有對應的 plot：sc.pl.umap、sc.pl.violin、sc.pl.dotplot、sc.pl.heatmap。

Plotting. Most tools have a matching plot: sc.pl.umap, sc.pl.violin, sc.pl.dotplot, sc.pl.heatmap.

📌
記憶法：pp = pre-process（前處理）、tl = tool（分析）、pl = plot（畫圖）。打 sc. 後按 Tab 看自動補全，比看文件快。 Mnemonic: pp = pre-process, tl = tool, pl = plot. Type sc. and press Tab — autocomplete is faster than docs.
