多體學整合

RNA + ATAC + 蛋白 + 代謝 — 把多種分子層次整合到同一張圖譜。

RNA + ATAC + protein + metabolite — integrate multiple molecular layers into one map.

概念

一、多體學定義

「Omics」= 全方位測量某層次的分子：transcriptomics、proteomics、metabolomics、epigenomics。

多體學整合分兩大類：

  • 水平整合 (horizontal)：多個樣本、相同 omics → 解 batch effect (Ch 14 的 Harmony / scVI)。
  • 垂直整合 (vertical)：同一群細胞，多種 omics → 找跨層級的關聯（基因表現 ↔ 染色質開放、ligand ↔ receptor、表型 ↔ 突變）。

本章重點是「垂直整合」：一個細胞同時量到 RNA + ATAC，怎麼一起分析？

"Omics" = comprehensive measurement of one molecular layer: transcriptomics, proteomics, metabolomics, epigenomics.

Multi-omics integration falls into two camps:

  • Horizontal: multiple samples, same omics → batch correction (Harmony / scVI in Ch 14).
  • Vertical: same cells, multiple omics → cross-layer relationships (RNA ↔ chromatin, ligand ↔ receptor, phenotype ↔ mutation).

This chapter focuses on vertical integration: how do you analyze RNA + ATAC measured from the same cell?

四、CITE-seq 流程

import scvi, scanpy as sc

# adata.X 是 RNA；adata.obsm["protein_expression"] 是抗體 counts
# adata.X = RNA; adata.obsm["protein_expression"] = ADT counts
adata = sc.read_h5ad("data/raw/cite_seq.h5ad")

scvi.model.TOTALVI.setup_anndata(
    adata,
    protein_expression_obsm_key="protein_expression",
    batch_key="batch",
)
model = scvi.model.TOTALVI(adata, latent_distribution="normal")
model.train()

# 整合後的 latent space
adata.obsm["X_totalVI"] = model.get_latent_representation()

# totalVI 還能去噪、預測未測蛋白
adata.obsm["protein_foreground_prob"] = model.get_protein_foreground_probability()
denoised_genes, denoised_protein = model.get_normalized_expression()

# 用整合 embedding 做下游
sc.pp.neighbors(adata, use_rep="X_totalVI")
sc.tl.umap(adata); sc.tl.leiden(adata)

# 同時看 RNA marker 與蛋白 marker
sc.pl.umap(adata, color=["CD3D","CD8A"])           # RNA
sc.pl.embedding(adata, basis="X_totalVI",
               color=["CD3_TotalSeqB","CD8_TotalSeqB"])   # 蛋白
